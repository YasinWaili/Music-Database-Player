INSTALL INSTRUCTIONS
npm install

LAUNCH INSTRUCTIONS
node server.js

TESTING INSTRUCTIONS
http://localhost:3000/login

How to Use:

Step 1:
- Download mp3 files of songs and insert them into the audio file
- Download Album cover pngs and insert them into the album_covers folder
- Insert the song names, album covers, and genre names into the sqlite3 database.

Step 2:
- Launch the server with Node.js

Step 3:
- Signup or Login to the webpage

Step 4:
- Search for the music you have downloaded and enjoy.
